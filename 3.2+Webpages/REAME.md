# Welcome Website Page

This HTML file creates a simple webpage with a main heading, a clickable image that links to the About page, a horizontal divider, and a text link that navigates to the Contact page.

## Tags Used
- `<h1>` – Used to display the main heading of the webpage.
- `<a>` – Creates a clickable hyperlink for navigation between pages.
- `<img>` – Displays an image on the webpage.
- `<a>` with `<img>` inside – Embeds an image inside an anchor tag to make the image act as a clickable link.
- `src` – Specifies the path to the image file.
- `<hr>` – Creates a horizontal line to separate sections.
- `<!-- -->` – Adds comments in the HTML code (not visible on the webpage).
