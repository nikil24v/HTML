# Birthday Invitation Page

This HTML file creates a simple birthday invitation using headings, an image, a list, and a link.

## Tags Used
- `<h1>`, `<h2>`, `<h3>` – Used for headings of different importance.
- `<img>` – Displays an image on the webpage.
- `src` – Specifies the image URL.
- `alt` – Provides alternate text for the image.
- `<ul>` – Creates an unordered (bulleted) list.
- `<li>` – Represents items inside the list.
- `<a>` – Creates a clickable hyperlink.
