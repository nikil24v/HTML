# HTML Nested Lists

This file demonstrates the use of nested lists in HTML using unordered and ordered list elements.

---

## Description

The HTML code represents a hierarchical list structure where lists are placed inside other lists to organize related items.

---

## Nesting in HTML

Nesting in HTML means placing one element inside another element.  
In this example, ordered (`<ol>`) and unordered (`<ul>`) lists are nested to create multiple levels.

Example:
```html
<ul>
  <li>B</li>
  <ol>
    <li>B1</li>
  </ol>
</ul>

Indentation in HTML

Indentation is the use of spaces or tabs to align HTML code for better readability.

Helps understand parent–child relationships

Makes code easier to maintain

Does not affect how the browser displays content

HTML Tags Used

<ul> – Unordered list

<ol> – Ordered list

<li> – List item
