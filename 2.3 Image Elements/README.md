# Cat and Dog Images

This HTML file displays headings and images of a cat and a dog using image URLs.

## Tags Used
- `<h1>` – Displays a main heading on the page.
- `<img>` – Used to embed images in the webpage.
- `src` – Specifies the image URL.
- `height` and `width` – Control the image dimensions.
- `alt` – Provides alternate text if the image fails to load.
