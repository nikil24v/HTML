## Key Concepts Demonstrated

- Use of the `<h1>` tag for main headings
- Use of the `<p>` tag to group textual content
- Use of `<br>` for line breaks within a paragraph
- Use of `<hr>` to separate sections of content
- Proper paragraph structuring for readability

## Void Elements
A void element is an HTML element that cannot have any content inside it and does not need a closing tag.

✅ Common Void Elements
<br> 
<img>
<hr>
<input>
<meta>
<link>
