# Movie Ranking Webpage

This project is a simple HTML webpage that displays a personal list of top movies.

---

## Description

The `index.html` file contains a movie ranking page titled **“The Best Movies According to Nikil”**.  
It lists the top 3 movies along with a short description for each one.

Headings are used to structure the page, and paragraphs are used to describe each movie.  
A horizontal rule separates the page title from the movie list.

---

## Contents of index.html

- Main page heading showing the title of the webpage
- Subheading indicating the movie ranking category
- List of three movies displayed using subheadings
- Short descriptions for each movie using paragraphs

---

## Technologies Used

- HTML5

---

## How to View

1. Open `index.html`
2. View it in any modern web browser

---

## Purpose

This project is created to practice basic HTML structure and text formatting using headings and paragraphs.
