# My Top 3 Favourite Websites

This HTML file shows a heading and an ordered list of three favorite websites with clickable links.

## Tags Used
- `<h1>` – Defines the main heading of the page.
- `<ol>` – Creates an ordered (numbered) list.
- `<li>` – Represents each item inside the list.
- `<a>` – Used to create hyperlinks to external websites using the `href` attribute.
