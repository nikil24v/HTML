# Angela's Cinnamon Roll Recipe

This project is a simple HTML webpage that presents a recipe for **Angela's Cinnamon Rolls**.

---

## Description

The `index.html` file displays a structured recipe page containing the recipe title, ingredients, and step-by-step instructions.

The ingredients are organized into two sections — **dough** and **filling** — using unordered lists.  
The preparation steps are presented in sequence using an ordered list.

---

## Contents of index.html

- Main heading displaying the recipe title
- Ingredients section with subheadings for dough and filling
- Unordered lists to list ingredients
- Instructions section using an ordered list for preparation steps

---

## Technologies Used

- HTML5

---

## How to View

Open `index.html` in any modern web browser.

---

## Purpose

This project is created to practice structuring content in HTML using headings, unordered lists, and ordered lists.
